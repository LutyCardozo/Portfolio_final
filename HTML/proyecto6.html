Proyecto 05 / recorrido creativo

Monte trail

Proyecto desarrollado para un obligatorio de Programación enfocado en la creación de un portfolio web utilizando inteligencia artificial como herramienta de apoyo. El objetivo no era únicamente generar código, sino comprender el proceso detrás de cada decisión, entendiendo cómo se construía la estructura, la lógica y la interacción del sitio. Para evitar repetir la temática del portfolio final, la propuesta tomó como concepto visual una carrera de trail running en Uruguay.

Alcance Interfaz interactiva
Terreno Diseño web
Ritmo Funcionalidad web
Meta Usar IA como herramienta, no como reemplazo

01

Planificación

El proyecto comenzó definiendo el concepto general del sitio y la forma de trasladar el universo del trail running a una experiencia web. Se trabajó sobre la estructura del recorrido, la navegación entre secciones y la manera en que cada pantalla podía representar distintas etapas de una carrera, manteniendo una identidad visual coherente y dinámica.

02

Largada

Se desarrolló la base funcional del sitio mediante HTML, CSS y JavaScript, utilizando ChatGPT como herramienta de apoyo para resolver dudas técnicas y optimizar partes del código. Más allá de generar resultados rápidos, el foco estuvo en comprender qué hacía cada fragmento y cómo interactuaban los distintos elementos dentro del sitio.

03

Resultado gráfico

El resultado fue una experiencia web interactiva inspirada en el trail running, donde el scroll se transforma en recorrido y cada sección funciona como una etapa dentro de la carrera. La propuesta combinó diseño visual, programación e inteligencia artificial para construir una navegación inmersiva, manteniendo siempre el entendimiento del proceso como parte central del aprendizaje.
