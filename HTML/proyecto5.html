Proyecto 05 / recorrido creativo

Poppi - Ecommerce

Proyecto realizado para un obligatorio de Programación, en el que se desarrolló un ecommerce funcional orientado a la organización y visualización de productos. La propuesta consistió en construir una tienda digital capaz de generar una experiencia clara, dinámica y funcional.

Alcance Ecommerce
Terreno Diseño web
Ritmo Funcionalidad web
Meta Ecommerce funcional

01

Planificación

El proyecto comenzó con la definición de la estructura general del ecommerce: páginas principales, catálogo, detalle de producto, navegación y funcionalidades necesarias para que la experiencia fuera clara y usable. A partir de esa planificación se organizó la información de los productos y la lógica de interacción.

02

Largada

En esta etapa se desarrolló la base funcional del sitio mediante HTML, CSS y JavaScript. Se trabajó con datos dinámicos para cargar productos, generar tarjetas, aplicar filtros y conectar las distintas secciones del ecommerce, buscando una navegación fluida entre el catálogo y el detalle de cada producto, usando ChatGTP como apoyo para resolver dudas técnicas y optimizar el código.

03

Resultado gráfico

El resultado fue una tienda online interactiva, con una estética visual definida y funcionalidades propias de un ecommerce. La incorporación de inteligencia artificial permitió sumar una capa adicional de interacción, integrando tecnología y experiencia de usuario dentro de una propuesta digital funcional.
