Proyecto 03 / recorrido creativo

JUGARTE + GARDNER

Proyecto de rediseño y unificación de identidad visual desarrollado para Jugarte y Colegio Gardner, dos instituciones educativas que compartían valores, equipo y espacio físico, pero se comunicaban como marcas completamente independientes.
El objetivo fue construir un sistema visual más coherente, flexible y reconocible, capaz de representar ambas propuestas desde una lógica común sin perder la identidad propia de cada una.

Alcance Identidad visual educativa
Terreno Diseño gráfico
Ritmo Sistema de marca
Meta Rediseño de marca

01

Planificación

El proyecto comenzó a partir de una observación clara: aunque Jugarte y Colegio Gardner compartían espacio, equipo y valores pedagógicos, sus identidades visuales funcionaban como marcas completamente separadas. El desafío consistía en construir un sistema gráfico más coherente, capaz de unificar ambas propuestas sin perder la identidad propia de cada una.

02

Largada

A partir de ese análisis, el trabajo se centró en desarrollar una identidad visual flexible y consistente. Se buscó generar un lenguaje gráfico común entre ambas instituciones, manteniendo una estética cercana, educativa y contemporánea que pudiera adaptarse a diferentes soportes y niveles de comunicación.

03

Resultado gráfico

El resultado final permitió fortalecer la percepción de unidad entre ambas instituciones, construyendo una identidad más clara, reconocible y profesional. El sistema visual se aplicó en distintas piezas y formatos, manteniendo coherencia gráfica sin perder el carácter humano y pedagógico del proyecto educativo.
